## Invoice Status

-   An invoice is always in the status tmp and therefore has an invoice number in the format TMP - xxxxxx. In this state, the invoice can be edited and changed at any time. Once the Send to client button is clicked, the invoice number changes from TMP to INV - xxx. This makes the invoice no longer editable. In addition, at this moment the invoice is assigned the current invoice date.

-   Additionally, the system should then automatically send the finalized invoice to the customer by email. We don’t have this in place yet, but it seems to be already prepared in the code.
